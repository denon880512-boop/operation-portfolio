# 用户运营作品集官网

基于 React + Vite 制作的个人作品集官网，已接入上传的首页视频、简历 PDF 渲染图和两张小红书账号截图。

## 运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 可替换内容

- 个人信息、经历、能力：`src/data/profile.js`
- 项目和账号案例：`src/data/projects.js`
- 简历下载文件：`public/resume.pdf`
- 首页视频：`src/assets/hero-cartoon.mp4`
- 简历视觉图：`src/assets/resume-about.png`
- 账号截图：`src/assets/xhs-panpan.png`、`src/assets/xhs-special-watch.png`
